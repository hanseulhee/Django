## Django

Django Study

---

##### 가상환경 생성
python -m venv[가상환경 명]

##### 가상환경 실행
mac - source[가상환경 명]/bin/activate <br>
window - source[가상환경 명]/Scripts/activate

##### 장고 설치
pip install django

##### 장고 프로젝트 생성
django-admin startproject [프로젝트 명]

##### 서버 실행 (프로젝트로 들어온 뒤)
python manage.py runserver
